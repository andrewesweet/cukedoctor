Cukedoctor documentation examples

All documentations are generated based on cucumber json output files generated during tests. The json formatter can be activated using:

  • cucumber features --format json --out reports (Ruby)
  • @CucumberOptions(plugins = {"json:target/cucumber.json"} ) (Java)
